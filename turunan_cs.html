Materi: Turunan Fungsi

1. Konsep Dasar & Interpretasi Geometris

Secara sederhana, turunan adalah Laju Perubahan Sesaat. Bayangkan speedometer pada motor; angka yang ditunjukkan pada detik tertentu adalah "turunan" dari posisi terhadap waktu.

Secara geometris, turunan merepresentasikan Gradien (kemiringan) Garis Singgung pada suatu kurva. Mari kita simulasikan!

Simulasi Interaktif: f(x) = x²

2. Aturan Dasar Turunan

Dalam memprogram kalkulator analitik atau AI, kita tidak menghitung limit manual, tapi menggunakan aturan paten (Power Rule & Sum Rule):

Power Rule (Aturan Pangkat):
Jika f(x) = xn, maka f'(x) = n · xn-1
Sum Rule (Aturan Penjumlahan):
Jika h(x) = f(x) + g(x), maka h'(x) = f'(x) + g'(x)

Kasus 1: Optimasi Biaya Server (Startup) +

Masalah: Sebuah startup Cloud Hosting memiliki fungsi biaya operasional per jam: C(x) = 2x² - 40x + 500 (dalam ribuan rupiah), di mana x adalah jumlah server aktif. Berapa server yang harus aktif agar biaya minimum?

Penyelesaian: Cari turunan pertama dan jadikan sama dengan nol (Titik Kritis).

  • C'(x) = 4x - 40
  • Syarat minimum: C'(x) = 0
  • 4x - 40 = 0 → 4x = 40 → x = 10

Kesimpulan: Startup harus menyalakan tepat 10 server untuk mencapai biaya operasional paling efisien.

Kasus 2: Pergerakan Karakter (Game Dev) +

Masalah: Posisi karakter NPC dalam game 2D dirumuskan s(t) = t³ - 6t² + 9t. Kapan karakter tersebut berhenti (kecepatan = 0)?

Penyelesaian: Kecepatan v(t) adalah turunan pertama dari posisi.

  • v(t) = s'(t) = 3t² - 12t + 9
  • Karakter berhenti saat v(t) = 0
  • 3(t² - 4t + 3) = 0 → 3(t - 1)(t - 3) = 0
  • t = 1 atau t = 3 detik.

Kesimpulan: Kode game akan menghentikan animasi jalan NPC pada detik ke-1 dan ke-3.

Kasus 3: Memperkecil Error (AI / Machine Learning) +

Masalah: Jaringan saraf tiruan memiliki fungsi error sederhana E(w) = w² - 4w + 10. Kita menggunakan algoritma Gradient Descent. Berapa arah gradien saat bobot w = 5?

Penyelesaian: Hitung turunan fungsi Error terhadap w.

  • dE/dw = 2w - 4
  • Substitusi w = 5: dE/dw = 2(5) - 4 = 6

Kesimpulan: Gradien positif (6), artinya komputer harus mengurangi nilai w (melangkah ke arah berlawanan) untuk memperkecil Error.
